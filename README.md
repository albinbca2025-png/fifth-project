🏦 SB Bank – Mini Banking System in C

A simple console-based banking application written in C that simulates basic banking operations like 
account creation, deposit, withdrawal, and balance checking.
-------------------------------------------------------------------------
🚀 Features

-🧑 Account creation with username
-🔢 Auto-generated account number
-💰 Deposit money
-💸 Withdraw money
-📊 Check account balance
-❌ Exit safely from the system
---------------------------------------------------------------------------------
🛠️ Technologies Used

-C Programming Language
-Standard Libraries:
-stdio.h
-stdlib.h
-time.h
----------------------------------------------------------------------------------
📂 How It Works

-User creates an account by entering their name
-System generates a random account number
-User is presented with a banking menu:
-Deposit
-Withdraw
-Check Balance
-Exit
-Operations are performed based on user input
----------------------------------------------------------------------------------
▶️ How to Run
-Step 1: Compile the Program
gcc bank.c -o bank
-Step 2: Run the Program
./bank
